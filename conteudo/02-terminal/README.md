# Módulo 02 — Terminal

Bem-vindo ao segundo módulo do **Linux Explorer**.

Depois de compreender os fundamentos do Linux, chegou o momento de começar a interagir diretamente com o sistema operacional utilizando o Terminal.

Este módulo foi desenvolvido para ensinar os conceitos essenciais da linha de comando de forma progressiva, explicando não apenas **como executar comandos**, mas também **por que eles funcionam** e **quando utilizá-los**.

Ao final deste módulo, você será capaz de navegar pelo sistema de arquivos, consultar a documentação oficial dos comandos, utilizar recursos que aumentam sua produtividade e executar tarefas básicas no Terminal com confiança.

---

# Objetivos do módulo

Ao concluir este módulo você será capaz de:

* Interpretar o Prompt do Terminal.
* Executar comandos básicos.
* Navegar entre diretórios.
* Listar arquivos e diretórios.
* Consultar a documentação oficial dos comandos.
* Utilizar recursos de produtividade do Bash.
* Executar múltiplos comandos em sequência.

---

# Como estudar este módulo

Para aproveitar melhor o conteúdo, siga esta sequência:

1. Leia o capítulo completo.
2. Execute todos os exemplos apresentados.
3. Realize a seção **Pratique**.
4. Resolva os desafios da seção **Explore**.
5. Avance para o próximo capítulo somente após compreender o conteúdo atual.
6. Ao finalizar o módulo, revise o **RESUMO.md** e utilize o **CHEATSHEET.md** para reforçar os comandos aprendidos.

A prática constante é a melhor forma de desenvolver confiança no uso do Terminal.
