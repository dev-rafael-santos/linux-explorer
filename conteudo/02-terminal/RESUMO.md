# Principais conceitos revisados

## Prompt

O Prompt indica que o Terminal está pronto para receber comandos.

Ele informa, entre outras coisas:

* usuário conectado;
* nome do computador;
* diretório atual.

Sempre que um comando termina sua execução, o Prompt volta a aparecer.

---

## Execução de comandos

Todo comando executado segue o mesmo fluxo:

```text
Você
↓
Shell
↓
Kernel
↓
Resultado
↓
Prompt
```

Compreender esse processo é mais importante do que decorar comandos.

---

## Navegação

O comando `cd` permite mudar o diretório de trabalho.

Também foram apresentados os conceitos de:

* diretório atual;
* diretório pai;
* diretório pessoal;
* caminhos absolutos;
* caminhos relativos.

---

## Listagem

O comando `ls` permite visualizar o conteúdo de um diretório.

Você também conheceu opções importantes como:

* `-l`
* `-a`
* `-la`

Essas opções serão utilizadas constantemente ao longo da documentação.

---

## Ajuda

O Linux oferece documentação integrada.

Você aprendeu a utilizar:

* `man`
* `--help`

Essas ferramentas permitem aprender novos comandos sem depender da internet.

---

## Produtividade

Você conheceu recursos que tornam o trabalho no Terminal mais eficiente:

* histórico de comandos;
* autocompletar com TAB;
* limpeza da tela;
* interrupção de comandos.

Pequenos recursos que economizam muito tempo durante o uso diário.

---

## Execução de múltiplos comandos

Você aprendeu três operadores importantes:

* `;`
* `&&`
* `||`

Eles permitem organizar a execução de comandos conforme a necessidade.

# Desafio prático

Imagine que você acabou de acessar um servidor Linux pela primeira vez.

Sem consultar os capítulos, tente realizar as seguintes tarefas:

1. Descubra qual usuário está conectado.
2. Descubra em qual diretório você está.
3. Liste os arquivos do diretório atual.
4. Entre na pasta `Documentos`.
5. Volte para o diretório Home.
6. Consulte o manual do comando `ls`.
7. Limpe a tela.
8. Utilize o histórico para repetir um comando.
9. Execute três comandos em sequência utilizando `;`.

Se conseguir concluir essas tarefas, significa que você já domina os principais conceitos deste módulo.
