# Visão Geral

Neste módulo você conheceu os fundamentos necessários para começar a estudar Linux com mais segurança.

Antes de trabalhar com comandos, arquivos e permissões, é importante entender alguns conceitos básicos:

- O que é Linux.
- Como um computador inicia.
- Qual é o papel do Kernel.
- O que são distribuições Linux.
- Por que o Terminal é tão importante.

Esses conceitos formam a base para os próximos módulos.

# Conclusão

Neste módulo você construiu a base necessária para compreender como o Linux funciona.

Ao longo dos cinco capítulos foram apresentados os principais conceitos que sustentam todo o restante da documentação.

Agora você sabe que:

- Linux é o Kernel do sistema operacional.
- Um computador passa por diversas etapas antes que o Linux seja carregado.
- O Kernel é responsável por gerenciar os recursos do computador.
- Distribuições Linux utilizam o mesmo Kernel, mas oferecem conjuntos diferentes de ferramentas.
- O Terminal é uma das principais formas de interação com o sistema.

Esses conhecimentos servirão como fundamento para todos os próximos módulos.
