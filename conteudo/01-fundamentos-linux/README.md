# Módulo 01 — Fundamentos do Linux

Este módulo apresenta os conceitos fundamentais para compreender o Linux antes de começar a trabalhar com comandos, arquivos, permissões e ferramentas de desenvolvimento.

## Objetivo do módulo

Construir uma base sólida sobre o que é Linux, como um sistema inicia, qual é o papel do Kernel, o que são distribuições e por que o Terminal é tão importante.

## Conteúdos

1. [O que é Linux?](./01-introducao-ao-linux.md)
2. [Como um computador inicia?](./02-como-um-computador-inicia.md)
3. [O Kernel Linux](./03-o-kernel-linux.md)
4. [Distribuições Linux](./04-distribuicoes-linux.md)
5. [Conhecendo o Terminal](./05-conhecendo-o-terminal.md)

## Ao final deste módulo

Você deverá ser capaz de:

- Explicar o que é Linux.
- Entender a função do Kernel.
- Compreender o processo básico de inicialização.
- Diferenciar distribuições Linux.
- Entender por que o Terminal é uma ferramenta central no Linux.
