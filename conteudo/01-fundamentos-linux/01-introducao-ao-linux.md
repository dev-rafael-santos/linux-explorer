# 01. O que é Linux?

> **Nível:** Iniciante
> **Tempo estimado:** 15 minutos
> **Pré-requisitos:** Nenhum

---

# Visão Geral

Se você pesquisar na internet o que é Linux, provavelmente encontrará respostas diferentes.

Algumas pessoas dizem que Linux é um sistema operacional.

Outras afirmam que Ubuntu é Linux.

Também é comum ouvir frases como:

* "Instale um Linux."
* "Uso Linux para programar."
* "Meu servidor roda Linux."

Mas afinal...

**O que realmente é Linux?**

Responder essa pergunta corretamente é importante porque praticamente todo o restante do seu aprendizado será construído sobre esse conceito.

Neste capítulo vamos entender, de forma simples, o que é Linux e por que tantas pessoas utilizam esse nome para se referir a coisas diferentes.

---

# Objetivos

Ao concluir este capítulo você será capaz de:

* Explicar o que é Linux.
* Entender por que existe confusão em torno desse termo.
* Conhecer, de forma resumida, a origem do Linux.
* Identificar onde o Linux está presente no nosso dia a dia.

---

# Por que isso é importante?

Imagine estudar mecânica sem saber a diferença entre um carro e um motor.

Provavelmente você aprenderia vários procedimentos, mas teria dificuldade para entender como tudo funciona em conjunto.

Com Linux acontece exatamente a mesma coisa.

Antes de aprender comandos, criar scripts ou instalar programas, é importante compreender o que realmente estamos utilizando.

Essa base facilitará o entendimento dos próximos capítulos.

---

# Conceitos

## O que é Linux?

A resposta técnica é simples:

> **Linux é um Kernel.**

Talvez essa resposta pareça estranha neste momento.

Não se preocupe.

Nos próximos capítulos estudaremos o Kernel em detalhes.

Por enquanto, basta entender que Linux é o componente central de um sistema operacional.

Ele é responsável por permitir que programas utilizem os recursos físicos do computador.

---

## Um pouco de história

Em 1991, um estudante finlandês chamado **Linus Torvalds** iniciou o desenvolvimento de um novo Kernel como projeto pessoal.

O objetivo era criar um sistema semelhante ao Unix que pudesse ser utilizado em computadores pessoais.

Com o passar do tempo, milhares de desenvolvedores passaram a colaborar com o projeto.

Hoje, o Linux é um dos projetos de software livre mais importantes do mundo.

---

# Como funciona

Imagine um automóvel.

Quando alguém diz:

> "Eu comprei um carro."

Na realidade, esse carro é formado por diversos componentes.

Existe o motor, a transmissão, os freios, a direção, as rodas e muitos outros sistemas.

O motor é uma das partes mais importantes do automóvel, mas ele não representa o carro inteiro.

Com Linux acontece algo parecido.

O Linux é o "motor" do sistema operacional.

Nos próximos capítulos conheceremos os demais componentes.

---

# Onde o Linux está presente?

Mesmo que você nunca tenha instalado Ubuntu em um computador, existe uma grande chance de utilizar Linux diariamente.

Ele está presente em:

* Smartphones Android.
* Servidores de internet.
* Plataformas de computação em nuvem.
* Smart TVs.
* Equipamentos de rede.
* Supercomputadores.
* Sistemas embarcados.
* Raspberry Pi.

Grande parte da infraestrutura da internet funciona utilizando Linux.

---

# Exemplo prático

Imagine que uma empresa deseja criar um servidor para hospedar um site.

Ela poderá utilizar uma distribuição Linux como Ubuntu Server ou Debian.

Essa distribuição será construída utilizando o Kernel Linux.

Perceba que:

* Linux é o Kernel.
* Ubuntu é uma distribuição.
* Debian também é uma distribuição.

Esse assunto será aprofundado em um capítulo específico.

Isso significa que bilhões de smartphones ao redor do mundo utilizam Linux como parte fundamental do sistema operacional.
