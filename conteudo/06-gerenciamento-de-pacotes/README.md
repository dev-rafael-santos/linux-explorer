
# Módulo 06 — Gerenciamento de Pacotes

# 🎓 O que você será capaz de fazer ao concluir este módulo?

Ao terminar este módulo, você conseguirá:

- instalar softwares com segurança;
- atualizar o sistema operacional;
- identificar a origem de um pacote;
- escolher entre APT, Snap e Flatpak;
- resolver problemas simples relacionados a pacotes.

## 📖 Apresentação

Neste módulo você aprenderá como o Linux instala, atualiza e remove programas.

Diferentemente de outros sistemas operacionais, a maioria das distribuições Linux utiliza gerenciadores de pacotes para instalar softwares de forma segura e organizada.

Você entenderá como funcionam os pacotes, os repositórios oficiais e os principais gerenciadores utilizados nas distribuições modernas.

Ao final deste módulo, será capaz de instalar programas, atualizar o sistema e administrar softwares utilizando boas práticas.

Ao concluir este módulo você será capaz de:

- atualizar o sistema;
- pesquisar softwares disponíveis;
