# Módulo 03 — Sistema de Arquivos Linux

Bem-vindo ao terceiro módulo do **Linux Explorer**.

Depois de aprender os fundamentos do Linux e utilizar o Terminal para navegar e executar comandos, chegou o momento de entender como o sistema operacional organiza seus arquivos.

Este módulo apresentará a estrutura do sistema de arquivos Linux, explicando a função dos principais diretórios e como eles se relacionam entre si.

Ao final deste módulo, você compreenderá a lógica por trás da organização do Linux e será capaz de localizar arquivos, configurações, programas e informações do sistema com muito mais confiança.

---

# Objetivos do módulo

Ao concluir este módulo você será capaz de:

* Compreender o conceito de diretório raiz (`/`).
* Identificar a função dos principais diretórios do sistema.
* Diferenciar diretórios destinados ao usuário daqueles utilizados pelo sistema operacional.
* Entender onde programas, configurações, logs e arquivos temporários são armazenados.
* Navegar pelo sistema de arquivos com mais segurança.

---

# Pré-requisitos

Antes de iniciar este módulo, recomenda-se concluir:

* **Módulo 01 — Fundamentos do Linux**
* **Módulo 02 — Terminal**

---

# Estrutura do módulo

## Capítulos

1. O diretório raiz (`/`)
2. Home e diretórios pessoais (`/home`)
3. Configurações do sistema (`/etc`)
4. Programas e bibliotecas (`/usr`)
5. Logs e dados variáveis (`/var`)
6. Arquivos temporários (`/tmp`)
7. Como o Linux inicia (`/boot`)
8. Dispositivos (`/dev`)
9. O Linux por dentro (`/proc` e `/sys`)
10. Aplicações opcionais (`/opt`)

---

# Como estudar este módulo

Para obter o máximo aproveitamento:

1. Leia um capítulo por vez.
2. Observe atentamente os diagramas apresentados.
3. Compare a estrutura do Linux com a do Windows quando indicado.
4. Execute os exemplos no seu computador.
5. Explore os diretórios utilizando os comandos aprendidos no módulo anterior.
6. Revise o conteúdo utilizando o **CHEATSHEET.md** e o **RESUMO.md** ao final do módulo.

---

# O que torna este módulo importante?

Grande parte das dificuldades enfrentadas por quem está começando no Linux está relacionada à organização do sistema de arquivos.

Ao compreender a função de cada diretório, você deixará de navegar "no escuro" e passará a entender por que cada informação está armazenada em determinado local.

Esse conhecimento será essencial para os próximos módulos, especialmente aqueles que abordarão manipulação de arquivos, permissões, administração do sistema e automação.
