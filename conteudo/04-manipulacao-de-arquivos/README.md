# Módulo 04 — Manipulando Arquivos e Diretórios

Bem-vindo ao quarto módulo do **Linux Explorer**.

Até aqui você aprendeu os fundamentos do Linux, conheceu o Terminal e entendeu como o sistema organiza seus arquivos.

Agora chegou o momento de começar a trabalhar com esses arquivos na prática.

Neste módulo você aprenderá a criar, copiar, mover, renomear, excluir, localizar e visualizar arquivos e diretórios utilizando o Terminal Linux.

Essas são tarefas executadas diariamente por administradores de sistemas, desenvolvedores, engenheiros de dados e profissionais de infraestrutura.

---

# Objetivos do módulo

Ao concluir este módulo você será capaz de:

- Criar arquivos e diretórios.
- Copiar arquivos e pastas.
- Mover e renomear arquivos.
- Excluir arquivos e diretórios com segurança.
- Visualizar o conteúdo de arquivos de diferentes formas.
- Localizar arquivos rapidamente.
- Compreender a diferença entre links simbólicos e hard links.
- Compactar e descompactar arquivos.
- Utilizar os principais comandos para manipulação de arquivos no Linux.

# O que torna este módulo importante?

Grande parte do trabalho diário no Linux consiste em manipular arquivos.

Saber navegar pelos diretórios é importante.

Mas saber criar, copiar, mover e organizar arquivos é uma habilidade essencial para qualquer profissional que utiliza Linux.

Ao concluir este módulo, você terá domínio sobre as principais operações envolvendo arquivos e diretórios.
