# Changelog

Todas as mudanças relevantes do projeto **Linux Explorer** serão documentadas neste arquivo.

Este projeto segue a convenção inspirada no **Keep a Changelog** e utiliza **Versionamento Semântico (SemVer)**.

---

## [0.2.0] - 2026-06-28

### Adicionado

* Estrutura completa do **Módulo 02 — Terminal**.
* 7 capítulos abordando os principais conceitos do Terminal Linux.
* Arquivo `CHEATSHEET.md` para consulta rápida dos comandos.
* Arquivo `RESUMO.md` para revisão do módulo.
* Novo padrão de capítulos com:

  * Visão Geral;
  * Objetivos;
  * Ficha do comando;
  * Prática guiada;
  * Exercícios "Explore";
  * Erros comuns;
  * Curiosidades;
  * Conclusão.
* README específico do módulo.

### Alterado

* Padronização da estrutura dos módulos.
* Atualização do `ROADMAP.md`.
* Organização da documentação para facilitar futuras contribuições.
