# Roadmap — Linux Explorer

Bem-vindo ao **Linux Explorer**!

Este roadmap apresenta a trilha oficial de estudos do projeto, organizada em módulos progressivos.

Cada módulo foi planejado para desenvolver seus conhecimentos passo a passo, começando pelos fundamentos e avançando até tópicos mais complexos.

---

# Como utilizar este roadmap

Recomenda-se estudar os módulos na ordem apresentada.

Cada módulo possui:

* README com a visão geral do conteúdo;
* capítulos organizados por assunto;
* CHEATSHEET com consulta rápida dos comandos;
* RESUMO para revisão dos conceitos.

Ao concluir um módulo, revise o conteúdo antes de avançar para o próximo.

# Filosofia do Linux Explorer

O objetivo deste projeto não é ensinar apenas comandos.

A proposta é desenvolver uma compreensão sólida sobre como o Linux funciona, permitindo que o estudante evolua de forma consistente e autônoma.

Sempre que possível, cada conceito é apresentado antes da prática, seguido de exemplos, exercícios e desafios.

O foco está em formar profissionais capazes de entender o sistema operacional, e não apenas memorizar comandos.
