# Código de Conduta

## Nossa missão

O **Linux Explorer** é um projeto educacional criado para tornar o aprendizado de Linux acessível, organizado e prático.

Nosso objetivo é construir um ambiente acolhedor, colaborativo e respeitoso para todas as pessoas interessadas em aprender e compartilhar conhecimento.

---

# Nossos valores

Esperamos que todos os participantes da comunidade:

* Demonstrem respeito e cordialidade.
* Compartilhem conhecimento de forma construtiva.
* Aceitem opiniões diferentes com maturidade.
* Incentivem iniciantes durante sua jornada de aprendizado.
* Mantenham uma comunicação clara e educada.

---

# Comportamentos esperados

São exemplos de boas práticas:

* Fazer perguntas de forma respeitosa.
* Ajudar outros estudantes.
* Corrigir erros de maneira educada.
* Compartilhar referências confiáveis.
* Sugerir melhorias para a documentação.
* Colaborar para manter o projeto organizado.

---

# Comportamentos inaceitáveis

Não serão aceitos:

* Ofensas pessoais.
* Assédio ou discriminação.
* Linguagem ofensiva.
* Ataques pessoais.
* Divulgação de conteúdo ilegal.
* Spam.
* Qualquer comportamento que prejudique a comunidade.

---

# Contribuições

Toda contribuição é bem-vinda.

Ao contribuir com o projeto, espera-se que o colaborador siga:

* este Código de Conduta;
* o arquivo `CONTRIBUTING.md`;
* os padrões definidos para a documentação.

---

# Resolução de conflitos

Caso exista alguma divergência entre colaboradores, recomenda-se:

1. Manter o respeito durante toda a conversa.
2. Buscar uma solução baseada em argumentos técnicos.
3. Priorizar sempre a qualidade do projeto.
4. Caso necessário, solicitar a mediação do mantenedor.

---

# Nosso compromisso

Nos comprometemos a manter um ambiente seguro, respeitoso e colaborativo para todos os participantes do projeto.

Acreditamos que compartilhar conhecimento é uma das melhores formas de fortalecer a comunidade de software livre.

---

# Obrigado!

Obrigado por contribuir com o Linux Explorer.

Esperamos que este projeto ajude milhares de pessoas a aprender Linux de forma clara, prática e progressiva.

Bons estudos! 🐧🚀
