# Checklist de Publicação

Antes de publicar um novo capítulo, verificar:

* Ortografia.
* Clareza.
* Exemplos funcionando.
* Laboratório revisado.
* Links funcionando.
* Referências atualizadas.
* Próximo capítulo definido.
* Estrutura seguindo o padrão do projeto.

---

# Encerramento de módulo

Antes de iniciar um novo módulo, verificar:

* Todos os capítulos concluídos.
* RESUMO.md criado.
* README do módulo atualizado.
* ROADMAP atualizado.
* Ideias futuras registradas.
* Commit de encerramento realizado.
